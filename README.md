# 글콘티 작성 마법사

시나리오, 극본, 대본, 자막, 텍스트 문서를 표준 글콘티 형식으로 변환하고 편집하는 브라우저 앱입니다.

## 주요 기능

- 원문 붙여넣기 또는 파일 업로드
- TXT, MD, CSV, TSV, JSON, HTML, SRT, VTT, RTF 직접 읽기
- DOC, DOCX, PDF, HWP, HWPX, ODT 브라우저 내 본문 추출
- 장면 안의 액션, 대사, 음향, 자막, 전환을 분석해 컷 단위 초안 생성
- 샷 크기, 카메라 움직임, 등장인물, 소품, 분위기 메모 자동 작성
- 글콘티 표준 항목 편집
- 작업 파일 저장/불러오기 (`.gconti.json`)
- 내보낸 HTML 글콘티 다시 불러오기 및 편집
- 최종 편집 가능한 HTML 글콘티 내보내기

## 사용

`index.html`을 브라우저에서 열거나 GitHub Pages로 배포해 사용합니다.
